# Personal Portfolio Website

# Overview

This is my **personal portfolio website**, built to showcase my skills, experience, and projects as a **Junior Software Developer / Full‑Stack Web Developer**. The website serves as a professional online presence where potential employers, collaborators, and clients can easily learn about me, explore my work, and get in touch.

The portfolio is designed with a strong focus on **clean UI**, **responsiveness**, **accessibility**, and **performance**, following modern web development best practices.

## Features

# Core Sections

* **Hero Section** – Introduction, role, and a clear personal brand statement
* **About Section** – Background, skills summary, and career focus
* **Quick Facts Section**

  * Experience (detailed, non-card layout for readability)
  * Location (card-based)
  * Availability (card-based)
* **Projects Section** – Selected works showcasing real-world applications
* **Contact Section** – EmailJS-powered contact form
* **Footer** – Social links, auto-updating year, and quick navigation

## Deployment Guide

### Option 1: GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Select branch (`main`) and root directory
4. Save and wait for deployment

### Option 2: Netlify

1. Log in to Netlify
2. Click **Add new site → Import from GitHub**
3. Select the repository
4. Deploy with default settings
